Mineral Identification in CRISM Data using 1D-CNN 🚀
This repository contains the complete implementation, documentation, and results for the automated mineral classification of Martian surface data.

📂 Repository Structure
The project is organized into specific folders to help you find exactly what you need:

/code: Contains all the Python scripts and Jupyter Notebooks used for the project.

    preprocessing.py: Spectral resampling (1501 bands) and denoising logic.

    cnn_model.py: The 1D-CNN architecture built with PyTorch.

    inference.py: Scripts to run the model on raw CRISM MTRDR cubes.

/results: A gallery of the output generated by the system.

    Includes Thematic Mineral Maps and Confusion Matrices showing the 93.07% accuracy.

/assets: Technical diagrams used in the project, including the ERD, Use Case, and Sequence diagrams.
